# AAdit
Simple Ascii Art Editor

# Installation
```
git clone https://github.com/skikozou/aadit
cd aadit
go get .
go install .
```

# Usage
### Start
```
aadit
// or
aadit {width} {height}
// or
aadit {width}x{height}
```

### Move
```
// use allow key ←→↑↓
```

### Write
```
// type character
```

### Console
```
ctrl + "/"
// type "help" and check commands
```
